# Sublayer-Rails

A Rails adapter for [sublayer](https://github.com/sublayerapp/sublayer), a model-agnostic Ruby AI Agent framework.

## Installation

```ruby
gem 'sublayer-rails', '~> 0.0.2'
```
`bundle install`

## Usage

`rails g sublayer:generator my_first`
